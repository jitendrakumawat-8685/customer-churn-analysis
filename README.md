# Customer Churn Analysis

Data analysis project on customer churn using Python, Pandas, NumPy, Matplotlib and Seaborn.

A complete data analysis project to understand why customers leave a telecom company.

## Project Overview
Analyzed 7000+ customer records using Python, NumPy, and Pandas to identify key churn patterns, then visualized insights using Matplotlib and Seaborn.

## Key Insights
- Overall churn rate calculated
- Month-to-month contract customers churn the most (42%)
- Customers with low tenure churn more frequently
- Electronic check payment method has highest churn rate
- Higher monthly charges correlate with higher churn
